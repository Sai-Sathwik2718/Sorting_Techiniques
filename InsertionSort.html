Welcome to Insertion Sort Evaluator!

***Performs Sorting using Insertion Sort Algorithm.***

In each pass first element in unsorted list is inserted in sorted list at an appropriate position. n-1 number of passes are required.

Insertion sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time. It is efficient for small data sets and works well for data that is already partially sorted.
